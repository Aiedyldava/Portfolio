# Hand Gesture Recognition

## Objective
I make this project for my thesis to play games without keyboard or controller, just making hand gesture to control it. 

## Methods
I'm using Tensorflow Lite Machine Learning 

## King of gesture
1. Up
2. Up Right
3. Right
4. Down Right
5. Down
6. Down Left
7. Left
8. Up Left
9. Pause
